# 💰 Personal Finance Tracker - SQL & Python Project

A simple but powerful personal finance tracking system built with **Python and SQLite**, designed to simulate real-world financial data analysis used in fintech and banking systems.

---

## 📊 Project Overview

This project allows you to track and analyze personal financial transactions such as income and expenses.

It demonstrates how raw financial data can be transformed into meaningful insights using SQL queries and Python.

---

## 🔧 Tech Stack

- Python 🐍
- SQLite (SQL database) 🗄️
- SQL Queries for data analysis 📊

## 🚀 Features

- Store financial transactions (income & expenses)
- Categorize spending (food, transport, entertainment, etc.)
- Calculate total income and expenses
- Compute overall balance
- Analyze spending by category
